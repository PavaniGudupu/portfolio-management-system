# Portfolio Management System

## Overview

A full-stack portfolio management system that allows visitors to view projects, skills, blogs, and contact information.

The administrator can manage all content through a protected dashboard.

---

## User Roles

### Visitor

- View portfolio
- View projects
- View skills
- Read blogs
- Download resume
- Submit contact form
- Open WhatsApp chat

### Admin

- Login
- Manage projects
- Manage skills
- Manage blogs
- View contact messages
- View analytics

---

## Features

### Authentication

- Admin Login

### Skills

- Update Skill
